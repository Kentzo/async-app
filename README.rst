Tests and CI
============

To run tests:

.. code-block::

    python setup.py test

Tests are run against CPython 3.6 (Windows and macOS) and above including dev and nightly builds (Ubuntu).
PyPy will be added as soon as all necessary 3.6 features will be implemented.
